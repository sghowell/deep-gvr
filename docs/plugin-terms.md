# deep-gvr Plugin Terms

The `deep-gvr` Codex plugin is distributed as part of the `deep-gvr` repository and follows the same open-source licensing and operational boundaries as the rest of the project.

## License

The repository is released under the MIT license in [LICENSE](https://github.com/sghowell/deep-gvr/blob/main/LICENSE).

## Intended Use

The plugin is intended for technically literate operators who want to run the `deep-gvr` verification workflow from Codex local.

It is not:

- a hosted managed service
- a guarantee of correctness
- a substitute for human review in high-stakes settings

## Operational Boundary

The plugin packages an existing workflow. It does not replace the underlying Hermes-backed runtime, and its successful operation still depends on your local environment, configured providers, and optional backend availability.

Use it only if you understand the runtime and provider dependencies involved in your chosen path.
