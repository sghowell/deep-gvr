# QEC Context

- Focus area: quantum error correction for fault-tolerant quantum computing
- Core concepts: surface codes, color codes, repetition codes, threshold theorems, decoders
- Early benchmark emphasis: rotated surface-code memory tasks under depolarizing noise
- Common verification split:
  - Tier 1 for literature grounding and logical consistency
  - Tier 2 for threshold or logical-error-rate claims
  - Tier 3 for formal claims about code properties or proof sketches
